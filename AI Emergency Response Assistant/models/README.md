# AI Emergency Response Assistant

## Overview
Offline AI system for emergency guidance using Gemma 4.

## Features
- Works offline
- Text + Image input
- Emergency alerts

## Tech Stack
- Python (Flask)
- Ollama (Gemma 4)
- HTML/JS frontend

## How to Run

1. Install Ollama
2. Run:
   ollama run gemma

3. Start backend:
   python app.py

4. Open index.html
